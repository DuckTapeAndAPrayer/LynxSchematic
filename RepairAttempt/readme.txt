This folder is about repairing a damaged rev control hub
based on a Lynx HIB rev 2.4c

What's inside:

story.txt
================
    The story behind the scenes, how the board got into this situation,
the investigation and the findings.
    You should read this file from bottom to top, as the updates will
be added on top of the file as new facts will be discovered.

photos folders
================
Photos of the board itself, with or without all components and
repair process snapshots, organised by the date of the story chapter.
